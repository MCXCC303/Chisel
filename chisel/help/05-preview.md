# 会话预览

在打包之前，你可以查看每个会话的具体内容，确保没有遗漏或多余。

预览从会话的**最后一条消息**开始加载（最近的消息），默认每批 30 条。点击**加载更多**可以向前翻阅更早的对话。

### 阅读指南

每条消息都有视觉标记来区分角色和类型：

- **青色竖线** `┃` 标记用户输入
- **紫色竖线** `┃` 标记助手回复
- **绿色框** 包裹文件编辑、命令执行和计划提交等工具调用
- **灰色斜体** 显示助手的思考过程
- **⚙ 图标** 标记文件读取等操作

由于会话内容可能很长，Chisel 采用了分批加载策略——只渲染你当前在查看的区域，超出范围的内容会被自动释放，避免内存膨胀。

> 如果你用过其他的 Agent 工具，想必你已经能看出来这个界面是从哪个 Agent 工具借鉴而来了。
