# 导出历史记录

导出是将一台机器上的 Claude Code 记录打包成可移植文件的过程。

### 第一步：选择要导出的项目

启动打包流程后，Chisel 会扫描本地 `~/.claude/` 目录，列出所有使用过 Claude Code 的项目。你会看到一个表格，包含项目名称、会话数量、历史条目数和数据大小。用
**空格键**勾选需要的项目，点击表头可以按列排序。

### 第二步：筛选会话

对于选中的每个项目，你可以进一步选择哪些会话需要打包，默认是全选。如果有某些不重要的对话想跳过，取消勾选即可。点击**预览**
按钮可以查看某个会话的具体内容，确认后再做决定。

### 第三步：指定输出位置

选择一个保存路径来存放生成的 `.tar.gz` 文件。可以直接输入路径，也可以点击文件夹图标用系统文件管理器选择。

### 第四步：开始打包

点击开始后，Chisel 会依次收集会话文件、文件历史、任务数据、命令历史，最后压缩打包。完成后可以一键打开输出目录。
