# 清除无效记录

当你删除、移动或重命名了一个项目目录后，Claude Code 中对应的历史记录并不会自动消失。这些「孤儿记录」占据着磁盘空间，但已经没有对应的项目了。

清除功能会扫描 `~/.claude/` 中所有项目路径，找出那些原始目录已经不存在的记录。

### 工作流程

**扫描**：检查所有项目的原始路径是否仍然存在于磁盘上。不存在的项目会被列出，包含会话数量、历史条目数和数据占用大小。

**选择会话**：对于每个无效项目，你可以查看和选择要删除的会话。默认会选中全部，也可以点击**预览**查看某个会话的内容。

**确认删除**：**这是一个不可逆的操作**。在正式删除前，会弹出二次确认页面，清楚地告诉你即将永久删除多少会话。

**执行清理**：删除操作会从以下位置彻底移除指定项目的指定会话数据：
- `~/.claude/projects/<encoded>/` 中的会话 `.jsonl` 文件
- `~/.claude/file-history/<uuid>/` 文件历史
- `~/.claude/tasks/<uuid>/` 任务状态
- `~/.claude/history.jsonl` 中的命令历史
- `~/.claude.json` 中的项目元数据引用

### 安全提示
- 删除前请务必使用**预览**确认会话内容
- 如果只是想迁移到新路径，请使用**打包**和**解包**功能
- 删除后数据无法恢复，请谨慎操作
